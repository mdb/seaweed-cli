# seaweed-cli

A Golang-based command line application for fetching surf forecast data from the [Magic Seaweed API](http://magicseaweed.com/developer/forecast-api).

# Usage

```
seaweed-cli --help
```

## Development

Running acceptance tests:

```
make test
```
